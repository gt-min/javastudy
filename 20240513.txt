  로컬레파지토리 : git
  1) 로컬레파지토리로 만들기
  2) staging  - 누구를 저장할 것인지 지정
  3) commit   - 저장

  작업 순서
  git config --global user.name 유저이름
  git config --global user.email 유저이메일
  git init
  git add a.txt
  git commit -m "commit message"
  git remote add origin https://github.com/gt-min/javastudy.git
  git push origin main
